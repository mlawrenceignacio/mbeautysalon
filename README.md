# MBeautyQueen Salon Management System — Web Frontend

This repository contains the **web frontend** of the **MBeautyQueen Salon Management System**, a modern salon platform designed to streamline client interactions, service bookings, and overall salon operations.

Built using **ReactJS** and modern web technologies, this application provides a responsive and user-friendly interface for customers to explore services, manage reservations, and engage with the salon online.

---

## Features

- User authentication & Google OAuth login  
- Online reservation system  
- Real-time chat (Socket.IO)  
- Client feedback & rating system  
- Promotions and service listings  
- Fully responsive modern UI  
- Real-time updates 

---

## Tech Stack

### Frontend
- **React.js**
- **Axios**
- **Socket.IO Client**
- **Zustand** (state management)
- **Tailwind CSS**

### Backend API
- RESTful API
- Hosted on **Render**
